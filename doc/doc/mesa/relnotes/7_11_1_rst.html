    Mesa 7.11.1 Release Notes / November 17, 2011

    ¶ ↑

    Mesa 7.11.1 is a bug fix release which fixes bugs found since the 7.11 release.

    Mesa 7.11 implements the OpenGL 2.1 API, but the version reported by glGetString(GL_VERSION) depends on the particular driver being used. Some drivers dont support all the features required in OpenGL 2.1.

    See the :doc:Compiling/Installing page <../install>` for prerequisites for DRI hardware acceleration.
