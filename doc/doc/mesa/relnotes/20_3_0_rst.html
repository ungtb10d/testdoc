    Mesa 20.3.0 Release Notes / 2020-12-03

    ¶ ↑

    Mesa 20.3.0 is a new development release. People who are concerned with stability and reliability should stick with a previous release or wait for Mesa 20.3.1.

    Mesa 20.3.0 implements the OpenGL 4.6 API, but the version reported by glGetString(GL_VERSION) or glGetIntegerv(GL_MAJOR_VERSION) / glGetIntegerv(GL_MINOR_VERSION) depends on the particular driver being used. Some drivers dont support all the features required in OpenGL 4.6. OpenGL 4.6 is only available if requested at context creation. Compatibility contexts may report a lower version depending on each driver.

    Mesa 20.3.0 implements the Vulkan 1.2 API, but the version reported by the apiVersion property of the VkPhysicalDeviceProperties struct depends on the particular driver being used.
